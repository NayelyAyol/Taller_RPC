<h1 align="center">🖥️ Taller RPC en Python</h1>

## Descripción
- Este taller consiste en la implementación de una calculadora y un conversor de grados usando RPC en Python mediante XML-RPC y gRPC.

- El sistema permite realizar operaciones matemáticas desde un cliente conectado a un servidor.

## Autor

Nayely Ayol

## Operaciones disponibles dentro de la Calculadora
- Suma
- Resta
- Multiplicación
- División

## Operaciones disponibles dentro del Conversor de grados
- Celsius a Fahrenheit
- Fahrenheit a Celsius

## Tecnologías usadas
- Python
- XML-RPC
- gRPC
